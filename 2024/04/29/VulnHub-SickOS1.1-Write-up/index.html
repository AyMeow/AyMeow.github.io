1.环境部署

  • 靶机来源：靶机详情页

  • 安装说明：

    • 下载下来解压获得ovf后缀文件，在”VMware -> 文件 -> 打开“中打开ovf文件即可部署成功。

  • 由于靶机详情页未说明该靶机的具体信息，考虑到接入现有公共网络中进行扫描存在信息混杂的情况，可以在”虚拟机 -> 设置 -> 网络适配器 -> 高级 -> MAC地址“中记录MAC地址，以便在后续打靶中更好的找到该靶机。
